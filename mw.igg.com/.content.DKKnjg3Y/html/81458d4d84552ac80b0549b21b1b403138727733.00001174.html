Myth War’s Open Beta is going to start very soon. We need to give our servers a unique name to make everyone playing the game very impressed. So we want your ideas for the new server names. If your ideas for the server name are adopted, we will offer you rewards.
Event Duration: From 9:00pm June 18th to 9:00pm June 22nd  EDT(GMT-4)
Requirements:
1.        The server name you offer us must fit the game’s style, content and background.
2.        The server name must be composed of only English letters.
3.        The server name must be simple but impressive.
4.        All participating players must offer names for more than 5 servers once and for all. These names must be related to each other.
Rules:
1. You can repeat posting the server names to reply. Trolling is forbidden. Or your qualifications will be cancelled.
2. If different players post the same server names, we will judge if the first post, including the name is valid. You can’t repeat posting replies of the same server names. Editing your posts is not allowed.
3. You can repeat posting with the same ID. You may win big rewards for the name you give to us.
4. We will release the winners' forum usernames after the end of event. Winners must send a private message to the appointed board manager to leave your account details and contact info. We are not responsible for any losses resulting from incorrect contact information.
5. We reserve the right to final interpretation.
Rewards:
Players will be given 300 silver ingots if the server names they offer are adopted.
