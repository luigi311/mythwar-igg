Caring Feathers Exchange

Level 50 Class Advancement, everybody knows this is a milestone for skill advancement in Myth War II. However, to achieve that, players have to beat Drowcrusher, which is not an easy job for newbie players to do. In order to help more newbie players complete this quest, the Myth War team is currently recruiting warm-hearted veteran players that are willing to share a little time and effort to aid those less fortunate than themselves. Those players who help newbie players finish this crucial quest will obtain Caring Feathers which can be exchanged for contribution points from an NPC. In addition, we will be holding a big Caring Feathers exchange event from which players can get even more cool stuff.

Event Duration:
Players can exchange with GM at pet market in server 1 from 8:00PM to 12:00PM on September 12th

Event Contents:
During this event, players can exchange caring feathers with the GM for delicious whammy eggs which can hatch into any of a few awesome pets at random. The Caring Feather exchange rate is as follows:

8 for a Whammy egg (80 gold ingots)
18 for a Peculiar Whammy egg (256 gold ingots)
25 for a Fantastic Whammy egg (329 gold ingots)

GMs will record the player's ID and number of feathers exchanged. At the end of the event, the player with the highest number of feathers exchanged will get an additional reward of Aurora Lion. If there is a tie (2 or more players with same number of feathers) the player that exchanged first will be the winner.

The fragrance always remains in the hand that gives the rose.
