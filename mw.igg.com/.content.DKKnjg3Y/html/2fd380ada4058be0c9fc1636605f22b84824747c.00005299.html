[Help] Myth War Client Support

Ground Rules
1. The only authorised URLs in any [Help] topic are:
*.igg.com
*.microsoft.com
www.free-codecs.com

Do not download any files from ANY url such as rapidshare or speedyshare etc

2. If you are at all in doubt about any file linked - Do not download it or follow the link and report it to Live Support

Installer will not run

1. Check the size of the download...

The size of install file Myth War 1.0.48.35 is 706 MB

2. Check your computer matches to the minimum specs required for the game

http://mythwar.igg.com/modules/download/index.php

3. Check for DirectX [ required ]

Start -&gt; Run
[Windows Key] + [r]

At the prompt type: dxdiag

This gives you a Diagnostics Tool, it'll tell you the version you have installed and any problems it encountered.

If it's not 9.0c update it from here -&gt;

http://www.microsoft.com/downloads/details.aspx?FamilyId=2DA43D38-DB71-4C1B-BC6A-9B6652CD92A3&displaylang=en

If there are errors identified, try and solve them. Google is a source for pretty much ANY piece of knowledge you desire.

4. Update your graphics card drivers

Usually the manufacturer of your card will have an updated driver or more recent, simply download and install

If it's an embedded or shared graphics card... check for drivers still or allow for more memory to be available...

5. Catastrophic Errors

In some cases, the installer may generate a collection of “Catastrophic Errors”, these are caused by an incomplete or corrupt download from a source. To solve, the easiest way is to download the client again, however, you could try asking Live Support to send you each of the files as they error up in the installer.

If they send the files, the files should be located in C:\Program Files\Mythwar_en\

The installer will you tell you a directory and the filename. IE: ani\skl221a.mda (don’t think that one exists...)

6. Re-download client?

If you're going to re-download the client, use a different source for it and preferably, when downloading large files, a download manager which allows start, stop and restarts to be completed is a good idea

Updater will not run or does not work ( Game does not connect )

This can occur for a number of reasons.

Update.* connects over a HTTP port or FTP (think HTTP) and so isn't usually filtered by ISP proxies or local firewalls... government proxies may filter FTP (as my work place did).

So, check your internet connection is up, I've tried to use the internet with a down connection

Check that your firewall, proxy, router and such allow the connection. If in doubt, the game uses ports 9006 and 9007 to run.

Try the most direct method of connecting to the internet as possible to see if the problem exists somewhere...

There are also plenty of cases where the update.* file has failed to work, sometimes a reinstall will cure this, others can copy the update.* from a working install to their own.

Failing this, download any patches released manually... and then run main.* instead of update.* ( requires either changing shortcut targets or opening the install folder )

Close all unnecessary programs and applications that use the internet on the form of a persistent connection to a single location. A persistent connection may include the use of a connected FTP client, an Internet radio stream, Automatic Updates etc.

For some reason, currently, MW refuses to establish a connection to the game server when one of the above criteria is true (given you aren't filtered).

Most applications will be easy enough to shut down and thus close their connections.

For internet explorer, you may find that you need to enter Task Manager.

[ctrl] + [alt] + [del]

Before Windows XP, it will start Task Manager automatically, on some versions or setups of Windows XP onwards, you may have to click the Task Manager option.

Pre-Windows XP.

Search out a process or entry to the list named "iexplore.*" and/or "iexplorer.*" Highlight it and click the "End Task" button.

Windows XP, Server 2003, Vista and subsequent versions

Click on the Process Tab.

Locate iexplore.* and click "End Process" say yes to whatever it asks you....

NB: Will add Automatic Update solution when I remember how to do it...

Pre-Game Issues

1. Fuzzy, "Static", or Blank screen...

&gt; Try update graphics card drivers, it's always good to have recent drivers!

&gt; Lower colour depth to 16 bit and try again.

Start &gt; Control Panel &gt; Display &gt; "Settings"
Right Click desktop &gt; Properties &gt; "Settings"

Download the Indeo Codec: http://www.free-codecs.com/download_soft.php?d=1686&s=188

3. Cannot load "door.dat" file

Delete the Door.dat file from the game directory, if it exists.

Run update.* and it should create a new Door.dat file.

If it doesn't, download the latest patch off the website and apply it again – the patches include Door.dat

4. Unable to open xxx\yyy(z).MDA

MDA problems have been quite persistent since MW began, perhaps because of compression errors, either way... MDA faults occur either during installation or patching. An MDA fault will not occur in a file that hasn't been altered.

If you're getting an MDA error when starting to play, and have just installed the game, the fastest option would be to reinstall or "repair" the installation.

If the game has been installed for a while, it would prove easier to re-patch the client to allow for a successful reinstallation of the MDA file.

To do so, open up the game directory ( default: C:\Program Files\Mythwar_en\ ) and open the Update.ini file.

The contents should resemble:
[quote:afc70dbe29="update.ini"]
[FTPConfig]
CurrentVersion =1.0.48.15[/quote]


To force update.* to redownload and "fix" the MDA fault(s)

The contents should be changed to the last previous patch version, if you're in doubt about what it is, check the website to see if you can locate it.

If not, you can alter the file to run through several updates by changing the 'c' value (usually patches are run through x.y.z... I thought, so now it's a.b.c.d to make it easier)

To make it easy, edit the contents to read:
[quote:afc70dbe29="update.ini"]
[FTPConfig]
CurrentVersion =1.0.48[/quote]


This will force update.* to run through all updates after the .48 version was released.

NOTICE: Always make a copy of this file incase it gets messed up when you do it.

5. Main.* has encountered an error and needs to close
Start -&gt; Run
[Windows Key] + [r]

At the prompt type: dxdiag

This gives you a Diagnostics Tool, it'll tell you the version you have installed and any problems it encountered.

If it's not 9.0c update it from here -&gt;

http://www.microsoft.com/downloads/details.aspx?FamilyId=2DA43D38-DB71-4C1B-BC6A-9B6652CD92A3&displaylang=en

If there are errors identified, try and solve them. Google is a source for pretty much ANY piece of knowledge you desire.

If you still have problems with the error try:
Start -&gt; Run
[Windows Key] + [R]

At the prompt type: sfc /scannow

You will be prompted for the Windows CD you used to install Windows.
If you legally and legitimately own the computer and a license, you should have no problems with this, simply stick the disc in and it’ll be happy to scan files and replace damaged or broken ones.

If you don’t have a CD – which even comes with OEM computers from Dell etc, that’s your loss on this suggestion and the alternative to this is to reinstall Windows.

During game run problems

Other than what was covered before connected (MDA ingame problems are solved in the same way as above)

There is one point to note currently. 1 or 2 updates ago, problems occured with persistent connections.

This has proven to be the problem with several people unable to connect to the game when opening after already using a persistent connection.

The other side of this is when you open the connection AFTER connecting to the MW server, then either the connection is lost for an internet stream, OR you close the connection, certain applications will force MW's active client to disconnect. In the case of Microsoft Internet Explorer, sometimes it will cause an error and be forced to close.

Be aware of these when playing

---
That's all solutions I've given for problems... hope it helps you guys get into the game...
