Technical FAQ - Client Support



All issues which relate to the clean running of mythwar.exe and the revised mythwar.exe* should be covered in the following several posts. When new issues are discovered, they will be added to the guide as I work them out properly. If you have a problem which is not covered by this Technical FAQ, please post a new thread pointing as much information about your problem, as best you can. Pictures are always a welcome help to assist.


This Technical FAQ covers:

  • Section 1: Download and Installation
  • Section 2: Startup and Connectivity
  • Section 3: Ingame Client Issues


Technical Specification

  • Operating System: Windows 98SE or higher.
  • Processor: Intel Pentium 3 - 800Mhz or equivalent.
  • Memory: 128MB or higher
  • Storage: Minimum of 1.5GB
  • Video Card: Support for 800 x 600 running at 16bit

Double the above specification for the recommended requirements to run MythWar.

MW will run under Linux on a similar spec PC (requirement of the Wine project) and Virtual PC's exist for Mac Users to use.



* Although you can't use the revised Mythwar client to play Mythwar (not very well at least), the same majority of principles apply to running the client.

Download and Installation



1. Download Fails
If you are having trouble with downloading the client, try using an alternative source or method. If you are using a Download Manager, such as FlashGet, try downloading with it. It is possible for Download Managers to corrupt your download as well as not using one.


2. Install Fails to run
If you can't the installer to run correctly, verify the size of download file (as current: 799mb). If your download is smaller, you may need to download again.


3. Catastrophic Failure
In some cases, the installer may generate a collection of â€œCatastrophic Errorsâ€, these are caused by an incomplete or corrupt download from a source. To solve, the easiest way is to download the client again, however, you could try asking Live Support to send you each of the files as they error up in the installer.

If they send the files, the files should be located in C:\Program Files\Mythwar_en\

The installer will you tell you a directory and the filename. IE: ani\skl221a.mda (donâ€™t think that one exists...)


4. Computer Stuff
Sometimes the installer will not run correctly, these issues were narrowed down to out-dated Graphics Card drivers and DirectX issues. It's always good to grab the latest drivers for your Video Card as they always work better than the set you have (if they're out of date). DirectX9.0c is required, though DirectX10 also works (DirectX 10 is Vista only)

Startup and Connectivity



1. Update.exe doesn't download the patch
In instances where the updater does not wish to download the latest or a number of patches, you will need to download the update(s) from the website: http://mw.igg.com/modules/download/gamepatch.php


2. Cannot load: Door.dat
Because dat files are crap and like to get corrupted this is sometimes a common problem. Usually update.exe will make a version of door.dat for you everytime it is run, however, sometimes it doesn't.

In these cases, you could borrow a friends door.dat or ask in Live Support.


3. Server is busy, unable to connect
Sometimes, the server is busy and you can't connect, although it's incredibly rare and usually it will specifically tell you its full.

However, usually it is all down to the client side.

Try 1 or more of the following until you work out what it is.
1. Turn off Anti-Virus and Firewall software.
2. Close things like Dreamweaver 8+, Streaming Radio stations, FTP clients etc.
3. Make as direct as possible connection to the interwebs as possible


MW uses ports 9006, 9007, 9998, 9999 and 10'000.

Ingame Client Issues



1. Fuzzy, "Static", or Blank screen...
Try update graphics card drivers, it's always good to have recent drivers

Lower colour depth to 16 bit and try again.

Start > Control Panel > Display > "Settings"
Right Click desktop > Properties > "Settings"

Download the Indeo Video Codec - http://www.free-codecs.com/download_soft.php?d=1686&s=188


2. MDA errors
MDA errors are often caused by not running the latest versions, however, they can also be caused by errors caused by the patch being applied incorrectly or incompletely.

In all cases, check that you have downloaded the latest patch.

You can always have the file copied to your computer from a friend, or try asking Live Support for a copy of the file.
