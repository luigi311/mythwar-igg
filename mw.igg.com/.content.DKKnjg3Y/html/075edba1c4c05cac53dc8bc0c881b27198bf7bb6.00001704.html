Mythwar II Closed Beta Test Launched

The long-awaited Closed Beta Test for Mythwar II Online (mw2.igg.com) has begun since 04:30 June 13(GMT-5)
We have released a new complete manual patch which is available here!
If your game client cannot be updated automatically, you can download and install the patch yourself.
You can choose any one patch at the at the Download Page(http://mw2.igg.com/download/client.php )  to download (Size: about 1.12M)
The installation operation is simple:
Download the patch program, run it and choose the Myth War II game client folder, then allow the program to install by itself. After that, everything will be done!
If you have any question, please feel free to contact us!

Thank you!
