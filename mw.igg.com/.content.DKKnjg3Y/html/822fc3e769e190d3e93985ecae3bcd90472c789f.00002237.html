How To: Making Big Titles

Hey everyone, got a little tutorial here on how to make your own BIG titles. So many ppl have asked me in game how to do it so im just gonna show everyone on here. First, in order to make a big title, u have to know the 'big word code'

Code  -  ~Z+

The code is the first thing typed when sending a message or changing ur title, dont not put any spaces in between the characters of the code. NOTE: when u type the code correctly, it will disappear and the next word after it will turn big as u type it
Example - "~Z+hello" (dont type the " ") looks like this..


Typing the code twice makes the words after it even bigger
Example - "~Z+~Z+hello" (dont type the " ") looks like this..


Typing the code a third time makes the words after it a lot bigger
Example - "~Z+~Z+~Z+hello" (dont type the " ") looks like this..


The code can b typed pretty much as many times as u want when u talk on local chat to use it, but when u make a title u can only type 17 characters. so dont use the code more than 3 times or else the words u want to make bigger wont fit. Now its time to use the new code u learned on ur title. Click on ur character at the top, then click "change title." when u see the title menu click inside the current title and erase ur current title completely (if u have a title). Now u type the code into the title box, then the words u want to make big. NOTE: when u have finished typing ur title, PRESS ENTER. do not hit confirm!

My title usually looks like this..
~Z+~Z+~Z+CHAZMAN

When u make ur own title u'll see a $ in front of ur title, i dont know how to get rid of it. if u find out how to then post it and i'll edit this post.
There u go, hope this tutorial helped explain y some ppl have those big titles! Any questions - post em.
enjoy your new titles!
