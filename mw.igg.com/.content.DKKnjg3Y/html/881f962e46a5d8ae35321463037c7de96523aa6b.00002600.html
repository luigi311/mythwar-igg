Notes

- All diamonds can be used on necklaces:
- Purple and orange diamonds can be also used on boots
- Remaining diamonds can be used on weapons
- Gems can't be used on weapons
- Purple Diamonds have double effect when used on boots
- All "Attacks" can either result in % rate or % damage / number of hits


Methods to obtain gems

- Mining: 1 g1 / min
- Miners: g3 gems (all in party)
- Hurricanes: g([Hurricane lvl] + 1) (random in party, leader excluded) - maximum: g11
- Papa Drow: g9 (rare)
- Lottery: up to g8
- Events: up to g13
- Winning Guildwar (up to g12)
- Guild Allotement: 15 x g6 for defeating Titans in time + 50 x g9 for top ranked guild (1 / guildmember)
- Wailing Stone: g5-g10
