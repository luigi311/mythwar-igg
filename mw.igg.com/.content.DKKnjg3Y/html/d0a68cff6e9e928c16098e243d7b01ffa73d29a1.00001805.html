Character / Armor / Font / Chat Appearance + Item Mall

Heya, first of all I want to say that Myth War Online is a very beautiful game with the hand painted backgrunds and everything, I couldn't stop playing this game when I first tried it out and it reminds alot about a console game of sorts. However nothing is never perfect and one can always improve in my opinion so I thought up some things that would attract alot more players and the main word is "Customize Ability" and here's my list of where you could improve:

1. Character Appearance - At the Character Creation Screen one could have options to be able to select different hair colors or skin colors. This would make every player feel unique and gamers love that

2. Armor Appearance - Each Armor Set lets say lvl 10, 20, 30.. etc. could have their own Looks, this would also allow for
players to feel even more Unique. One could also have so that one is allowed to pick a "Color in Stock" for the Armor. With "Color in Stock" I mean that for example there is no way that a character can buy a Pink armor if another player on the same Server already have a Pink armor of the same armor set, it's just an idea of mine or if one in another way could Mark an Armor as Unique for the Individual Player. (Maybe with some additional armor art)

3. Font Appearance - I've never made a game but I don't think it would be too hard to implement and allow the players to pick a font of their liking in the System Settings. Would get a *thumbs up* from me if it was implemented

4. Chat Apperance - In Full Screen Mode when in a battle the "open chat" is overlapping the Character Info/Action Box, I think this is slightly annoying, if you could permanently place it somewhere like below the Character/Pet Battle Boxes that would also be awesome.

When I wrote these suggestions I have no idea how old this game is and it was never my intent to upset anyone if I did by posting this

Only other last thing I could think of is:

5. Item Mall - I just took a peek in it for my first time by clicking on the Aries Pack on the Game Start-up Screen, when I later log in on the website and try find the Aries Pack in the Item Mall I can't find it so my Suggestion is to make the Item Mall more syncronized with the Advertisements on the Game Start-up Screen and even re-organize a bit to make things easier to find with maybe new Categories. For example:

Adding Content:
- Update Item Mall to include New content just before
- Updating the Game Start-up Screen

Removing Content:
- Update Game Start-up Screen before
- Removing content from the Item Mall

I hope this was of value to the Game Developers as well as an inspiration for other players to make more Suggestions

That's all I had to contribute after my first 2 days of game-play and I'll be sure to add more Suggestions or even Guides if I feel there's not better guide on the Forum already, as I stumble across them.

Regards, Techlari
