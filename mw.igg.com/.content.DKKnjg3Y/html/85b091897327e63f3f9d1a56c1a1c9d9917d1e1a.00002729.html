Q) How can I create a character?
A: 1) Login to the game;
2) Select “Create New Character”;
3) Select a character you like;
4) Select the character’s sex;
5) Create a name and press “Enter”.

Q) What is the chat channel?
A: There are several modes of chatting in Myth War:
World: Talk to everyone in Myth War game.
Market: Talk to people on the same server.
Guild: Talk to people on the same guild/ country.
Private message: To the designated characters/players.
Party: Talk to the same group of people.
Local: Talk to people on the same screen.
