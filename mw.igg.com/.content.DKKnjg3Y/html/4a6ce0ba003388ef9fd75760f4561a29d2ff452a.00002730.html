Program Related
Q) If during downloading or updating of Myth War client, a disconnection occurs, what can I do?
A: Please check your internet connection and firewall settings and if it still doesn’t work, please check the Myth War website for the game server status, it might be turned down temporarily for maintenance.
2007-02-28
Q) If during downloading or updating of Myth War client, there is insufficient hard disk space, what can I do?
A: Please free up hard disk space prior to installation to ensure that there is sufficient space to install Myth War.
2007-02-28
Q) I have other applications running while running Myth War, will this affect the game stability in anyway?
A: Yes, there are a few programs that will cause clashes and system errors with Myth War, such as 'RealPlayer'. Please make sure that RealPlayer is not running while playing Myth War.
2007-02-28
Q) After installation, I can only hear the sound but cannot see the pictures, what can I do?
A: Please right click your destop and then choose properties to open "Display Properties", click "Settings"" then set Color quality as Medium (16bit) and Screen resolution as 1024 by 768 pixels. If still fail, please restart your computer.
2007-02-28
Q) Which server should I choose to play in?
A: When you launch the game you can choose any one server you would like to play in, the severs’ time settings are different, please check out the most suitable one for you.
2007-02-28
Q) After I entering the user name and password, there is no response from the server, what can I do?
A: Please check your internet connection and firewall settings to make sure they are correct. If you are still unable to launch the game; please check the Myth War Official website for the game server status.
2007-02-28
Q) How can I keep my version of Myth War updated to the latest version?
A: Upon launch, the game will detect if there are any new patches available. If there is a new update, the patch will be downloaded automatically. Alternatively, you can download the patch manually from the Myth War website.
