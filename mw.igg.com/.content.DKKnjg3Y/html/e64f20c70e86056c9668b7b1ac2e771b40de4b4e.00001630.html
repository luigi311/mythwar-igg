Closed Beta is designed to catch as many bugs as possible as well as test server strain.

Usually all data is wiped after a short (in this case, 10 day) closed-beta stage.

An Open Beta stage will follow, where everyone can play, and most likely, the data will not be wiped.

This is all based on information I have gathered today, so correct me if I'm wrong; however, I believe this is the case.

Just have to be patient.

[EDIT] Questions about beta can probably be answered via article.  http://en.wikipedia.org/wiki/Software_release_life_cycle#Beta
