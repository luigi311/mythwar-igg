12# A A A Post at 2008-6-18 02:17 PM  Show author
Just a few notes. Gr on pet is not specific to lvl of pet. Obvious I know, but worth mentioning for new players.
The requirement for using pets is obtained by completing the soul stone quests. EG: To use a Hell Cat you need to have "Wanderer Title" This is rewarded for completing the the 1st Soul Stone quest.
Some pets cannot be captured. These include HB'S, Evil Guards, Spirits in Lairs, Bosses (I fooking wish,lol). The reasons for most are obvious, but some are going to be intended event prizes, mall pets only, and bombing rewards. So there would be little point letting you catch a free one.
Mystical Fruits. There is a lvl requirement to use a MF on a pet. So bank it and wait till pet lvls up a bit. This means training to some extent at low levels is now a must rather than feeding with fruits.
Capturing a pet uses MP's. So the more MP's you have the more turns you can have at capturing. This will be an advantage to some races and not others.
The number of pets you can hold is level specific. The higher the level the more pets you can hold. So no point trying to capture a pet if your full.

Anyone can correct any of this or add to it please do. Still early days, so just reporting as I find. If I didn't drink so much when playing I'd probably remember more
