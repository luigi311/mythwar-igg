1. Introduction to the Chatting System

About the Chat System
The purpose of the chat system is to bring the players closer together and to provide a means of communication. You can type in your messages at the bottom left corner of the game screen. Upon pressing enter, your message will be displayed in the chat window. The button at the bottom left corner allows you to toggle between the messages outside of the game screen or inside the game screen. You can choose the channel to talk in based on the audience you want to reach out .to. Currently we have a few different modes : Local, Party, PM, Guild, Market and World. Emoticons are also enabled in the chat system. To use them, just key in the ‿‿character and follow by the number from 0 to 99

About Chat Interface

  • The chat interface comprises of the following : chat display, function keys, input display and channel list display.
  • Chat display : to display the message posted by players.
  • Messages are displayed in this format : [Channel][player’s nick][message content]
  • When the game is in full screen mode, the chat display can only be on the screen.
  • When the game is in window mode, the chat display can either be on the game screen or in another window.
  • When the game is in full screen mode, you can only choose to have the chat display either on the game screen or choose to turn off the chat display.
  • The system will be able to auto adjust the size of the chat display based on the following:
  • When the monitor is set to 800 x 600 pixel, and the chat display is set to display in another window, the player will not be able to see the game screen.
  • When the monitor is set to 1024 x 768 pixel, and the chat display is set to another window display, the game screen will be changed to top left hand corner at co-ordinates (0,0). The chat display can be dragged to any of the 4 side of the game screen. When the chat window is at either the left or right of the game screen, the chat display will be fixed at 200x600 pixels. When the chat display window is either at the top or bottom, the chat display will be at 800 x 138 pixels.
  • When the monitor is set at 1280 x 960 pixels, and the chat display is set in another display window, the chat display by default is at the side of game window (it can be drag to other 4 sides of the game screen), the chat window can be adjusted between 240x600 pixels to 480 x 600 pixels. When the chat display is at the top or bottom of the game window, the size of the chat display window can be set from 800x165 to 800 x 330.
  • The system will set the game display window to be the parent window and the chat display window to be in the child window. Hence the chat window does not have any control button. When the game window is being closed, the chat window will be closed too.
  • Chat input method: when there’s no other input window being open, the message box at the bottom left hand of the game window will be the input window by default. Players just have to click there and type in their messages and press [Enter].
  • Chat function keys:
    - Display switch: toggle between in screen display or chat window display.
    - Scroll up: to scroll the message up
    - Scroll down: to scroll the message down
    - Chat history: Display the history for up to 1024 messages.
    - Emoticons: Display the emoticons available to choose
    - Channel: Select the channel to display your message.
    - Maximize the chat window
    - Message window
    - History: the system is able to store up to 16 lines of message.

2. Introduction to Channels

Channels
By default, the font colour is set to white. The nickname of the player will precede the message. The channel colour will be in a different colour.

Local Channel

  • The chat dialog box will appear at the head of each character.
  • All players on the same screen will be able to view the message.
  • There are no level limits to using the chat system.
  • The messages will stay on the screen for 3 seconds.

Party Channel

  • The chat dialog box will appear at the head of each character.
  • All players belonging to the same party will be able to view the message.
  • There are no level limits to using chat system.
  • The message will stay on the screen for 2 seconds.
Guild Channel
  • The chat dialog box will not appear at the head of each character.
  • All players of the same guild will be able to view the message.
  • Only players in a guild will be able to use this function.
  • The message will stay on the screen for 3 seconds.

Market Channel

  • The chat dialog box will not appear at the head of each character.
  • All players at the same screen will be able to view the message.
  • There are no steadfast rules to using this channel but messages should adhere to the following syntax : ‘sell‿ ‘buy‿ ‘purchase‿ ‘offer‿and ‘request‿along with the name of one of the items.
  • When the market channel opens, right click the items you want to sell off, the system will auto input the item name. When other players see it, they’ll be able to click on the item and get more information on the item.
  • The messages will stay on screen for 30 seconds.
  • In the event that the GM finds out any abuse of the system by players, warnings will be given out.

World Channel

  • The chat dialog will appear at the head of each character.
  • All players on the same screen will be able to view the message.
  • There are no level limits to using chat system.

Private Channel

  • The chat dialog will not appear at the head of each character.
  • All players at the same sever will be able to use the system.
  • There are no level limits to using the chat system.
  • Right click on the player to chat with and he’ll be able to see your message.
  • The message will stay on the screen for 2 seconds.
System Channel
  • System messages sent out by the GM will be in the colour RED.

Appendix

  • The chat dialog above the character’s head will not be able to follow the character and will disappear after 5 seconds.
  • The maximum number of chat dialogs that appear above the character’s head is 3.
3. Introduction to Chat Commands

Channel Colour Commands (not case sensitive)

#R - red
#G - green
#B - blue
#K - black
#Y- yellow
#W- gray
#S - blink
#Cxxxxx - 6 numbers to customize your own colour
#U - underlined
#O - bold

4. Chat System

Function Outline

  • Works in a similar way to MSN Messenger. The main purpose is for players to communicate amongst each other.
  • Receive messages from a buddy in a private window.
  • There are 3 categories : buddy, acquaintance and trading.
  • Functions includes private messaging, mass messaging function, buddy online function, buddy sequence, auto reply, player blocking, online players search, history search, etc‿

Buddy List

  • The buddy sequence order of appearance is as follows : those who are online first, followed by degree of friendship and then colour differentiation.
    - pink is for couples
    - dark blue is for disciples
    - purple is for brothers
    - black is for other online good friends
    - gray is for others
  • To add a buddy, you may search for them using their nick or ID or you can add your friends from the game or add them from your acquaintance list.
  • To mass email : double click a buddy, type in your message and click on the ‿‿beside the buddy’s nick to add other buddy.
  • To check the information on a character, click on the player’s nick in the buddy list.
  • You can break off a relationship.
  • You may put a block on a player.
  • You can have a private conversation.
  • There is a history log.
  • When players are added to your buddy list, the degree of friendship will start accumulating.
Acquaintance List
  • The names in the acquaintance lists are all in black.
  • You can add acquaintances by searching, clicking on their character or accepting their private chats.
  • Acquaintances will be deleted when either one goes offline, is manually deleted or is added as a buddy.
  • The GMs will always be in the top position in the acquaintance list.
  • To reveal a player’s ID, click on the character.
  • To send a message: left click on the player’s nick and the input window will appear. Type in your message and press enter or click confirm.
  • To check on the player’s character: right click on the player’s nick, the nick, ID, level, title, guild, race, friendship and relationship will appear. You can also add as friend, acquaintance, update, delete, block and PM.
Sending messages
  • Messages can be classified into different channels.
  • There are no level limits.
  • Sending message interface : recipient nick, add as friend, delete, block and PM
  • Receiving message interface : sender's nick, add as friend, delete, block and PM
  • The messages can be sent to offline players. The system will auto save the message and the players will receive their messages when they next come online.
  • Players can send mass messages to other players.
  • The maximum number of characters is 256 words.
  • A maximum of 250 offline messages can be stored.

Block

 This function is for players who do not like to receive message from certain players.

  • The system will ignore all messages sent to the players by the particular player.

Online Player Search

  • You can search for players using their nick or ID.

Message History and Recording

  • There are 2 types of history: chat history and message history.
  • Chat history can store up to 1024 lines.
