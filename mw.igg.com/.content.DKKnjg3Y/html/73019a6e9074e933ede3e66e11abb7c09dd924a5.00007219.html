 Solution for DOOR.DAT | Solution for BUSY SERVER
find the solution on this reply!
To solve the problem DOOR.DAT
Go to the MythWar(_en) folder ( on my swedish computer the whole adress is: C:\Program\MythWar_en )
Follow following instructions after that you have opened the MythWar folder:

1. Right click and go to NEW. Then make a new NOTE PAD.
2. Leave the NOTEPAD empty and go to Archive / Options.
2. 2. On some computers the Archive / Options window dont exists then just click SAVE AS...
3. Change the name on SAVE AS... to door.dat ( NOT DOOR.DAT.TXT )
4. Click SAVE or OK
5. Open the game and everything should work


<-- End of this solution -->________________________________
To solve the problem BUSY SERVER
Lets go direct to the instructions!


1. Download this patch ( 32 ) ftp://66.223.51.39/pub/1.0.48.32/bin.zip and unzip/extract it in the MythWar folder ( instruction for finding the folder is in the first solution )
2. When you unzip/extract the patch Windows will ask you if you want to overwrite the already existing file then just click YES/OK on all of the requests from Windows
3. Make sure you have the DOOR.DAT
4. Start the game and enjoy!


NOTE: Even if there is a newer version of the game still download that patch and add it, later the game will update when you open the game.
You can also try to find the url to the newer patch on the official gamesite.
