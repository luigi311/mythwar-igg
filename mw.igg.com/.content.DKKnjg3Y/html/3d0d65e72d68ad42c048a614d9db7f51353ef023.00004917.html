Internet Safety

Stage 1: Limited Vs Administrator Accounts
Note: Windows Vista uses a "UAC" to annoy you about installs, but this still applies somewhat
Note2: This doesn't apply to anything older than Windows XP... I don't think (correct me if I'm wrong)

SpyWare and Adware all love to bury themselves deep within files and folders which Windows loves too.

A Limited account is somewhat heavily restricted on what it can and can't do. While sounds rubbish, it's actually quite brilliant.

A Limited account cannot modify read only files, it can't register DLL's, it can't put stuff in the Windows directory, they can't install a lot of programs etc.

Why is this good? Simple. If you can't do it, neither can the vast majority of rubbish that will attempt to infect your PC.

BUT, The downside is that the Limited Account isn't "cool" enough for some programs - specifically those which like to update themselves every few days...

For that, you may have to run them as an Administrator, but for the sake of 10 minutes, you're not making a big imprint.

While it may sound annoying still that you can't install programs, if you right click the installer, there is nearly always a "Run As Administrator" option available, hit it, enter password and it may install.

Again, some things can only be run as Administrators. So while this solution will be good for some, it won't be good for others. Try it and see how it goes - you could upgrade and downgrade your account priveledge as needed if you share the computer.


Stage 2: Safe Internet Usage

1. - If you don't know a thing about the link, don't click it.
2. - If you recieve an attachment which has .bat, .*, .pif, .vb[s], or contains multiple '.' don't click it. Chances are it's dodgy.
3. - If you recieve a somewhat suspicious email from a friend, and it has a suspicious looking attachment - don't open it. If you think it is, it probably is.
4. - Don't click on Advertisements - if you like the look of something, stick the URL in yourself.
5. - Be warey of Javascript boxes which ask you to do things. If you can right click the box and get a menu up - hit the little x in the corner of the message. Do not press OK
6. - This will upset some people but, "Adult" sites... are FULL of rubbish you want. There are some which aren't (apparently, probably PPV) but nonetheless, don't go there.
7. - Don't go looking for cracks, hacks and/or illegal software.
8. - "Free Instant Messaging Smilies" bad idea.
9. - Use P2P networking - Even stupider idea to look for Anti-Virus and Firewalls on it.
10. - Gambling outfits... don't go there.
11. - Don't bling your phone out with stuff off the net - it's too expensive, it's rubbish anyway, and the website is littered with spyware.

Note: 7-11 were provided by a MS-MVP posting on an Anti-Spyware site.

Stage 3: Windows Update

This is a large one really, always make sure you are running all the latest High Priority Hot Fixes from the Windows Update site. Microsoft get informed of serious holes in their software and they patch them - hotfix. If you don't apply them, you're purposely leaving a much larger target area and in some cases - a big neon sign saying "Here Am I!".

There's also a new(sih) dealy there now called "Microsoft Update". Unlike "Windows Update", this new(ish) version takes a bit longer, but finds updates for all the Microsoft programs you run, One-Care, Defender, Office and obviously, XP/Vista (nothing else is supported I don't think).

Stage 4: ActiveX -
ActiveX is the internet's application - as created by Microsoft.

Open Internet Explorer - Go to: Tools &gt; Internet Options.

From here, Pick the Security tab, highlight the Internet zone, then press Custom Level.

In Custom Level, there's a lot of options, each alters your security settings. (surprise surprise)

Find the ActiveX section of the lists.

Set: Download signed and unsigned ActiveX controls to "rompt" - asks if you want to run the control or not.
Set: Initialize and Script ActiveX controls not marked as safe to "Disable" - disables the control outright.

Stage 5: Anti-Virus

Straight forward. Install a good one, Buy it legitimately or not at all; AVG and Avast are free providers of AV Software, as are plenty of other companies.

An interesting comment picked up from the net is, if you think you're infected and don't have an Anti-Virus, go to the website of an Anti-Virus manufacturer - IE: Symantec or Trend Micro and they often have a free web-based scanner to use, clear the virus and then install. Many people believe installing makes it worse...

Stage 6: Firewalls

As above really. Install a good one. The Windows Firewall is somewhat acceptable, however, unlike third-party Firewalls, it doesn't offer 2 way protection for traffic, only Inbound traffic is disturbed.

Kerio seem to provide a very popular and highly recommended Firewall. Though, as aove, there are free ones!

Stage 7: Anti-Spyware

You can have enough of them. Each has advantages, each has weaknesses. Having too many will make them all annoyed though each one is capable of finding things others aren't.

Ad-Aware: Provides a free version, finds plenty of stuff, is recommended by Google, however the Ad-Watch feature and scans do take quite a bit of your resources away.
SpyBot: S&D - Lots of people love it (I've never used it...), quite a few Anti-Spyware sites promote it, probably because it's small in download size. the TeaTimer feature will take up resources too.
PrevX2.0 - I wouldn't be surprised if nobody had heard of it, it's rather low-key, but brilliant. The software is free... until it protects you from something. the description says you don't have to pay at all, it'll still tell you about things, you'll have to solve them manually though.
Defender - Microsoft's alternative, and a handy alternative it is too, somewhat more "meaty" than it's fellow brethren, Defender provides real-time protection on numerous ways that you can get infected.
SpywareBlaster - Javacool's entry to ASW Protection is pretty good too, focusing on ActiveX-based threats and such.
SpywareGuard - Again by Javacool and can complement the previous application too.

Which applications, some or all, you choose is up to you. They should all work together and be happy in doing so. As before, Each has advantages, Each has weaknesses.



Extra Notes

While employing a bunch of software to protect you from all sorts of garbage floating on the net, you will feel a performance hit - most of the above applications will all run in the background, although, they can all be turned off if preferred.

If you choose to turn them off, using MSCONFIG or the applications themsevels, you will have to remember that you'll have to run the programs manually to find out if you've been silly for a few seconds too long.
Most people would rather be certain they're miserable than risk being happy.
